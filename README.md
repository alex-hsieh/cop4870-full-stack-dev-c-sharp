the application should include two model classes with the following specifications:
- ToDo logic: Id , Name , Description , Priority , IsComplete , ProjectId

- Create a ToDo
- Delete a ToDo
- Update a ToDo
- List all ToDos
- Create a Project
- Delete a Project
-  Update a Project
-   List all Projects
-    List all ToDos in a Given Project
